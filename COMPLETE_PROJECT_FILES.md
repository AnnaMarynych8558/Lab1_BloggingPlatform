# Повні файли Flask Blog проєкту

## Структура папок:
```
flask-blog/
├── app.py
├── main.py
├── models.py
├── routes.py
├── forms.py
├── README.md
├── LICENSE
├── .gitignore
├── dependencies.txt
├── DEPLOYMENT.md
├── templates/
│   ├── base.html
│   ├── index.html
│   ├── login.html
│   ├── register.html
│   ├── create_post.html
│   ├── edit_post.html
│   ├── post_detail.html
│   ├── profile.html
│   ├── create_category.html
│   └── error.html
└── static/
    └── style.css
```

## Інструкції:
1. Створіть папку `flask-blog`
2. Скопіюйте всі файли згідно зі структурою
3. Перейдіть до наступного повідомлення для отримання змісту кожного файлу
